# safe-url-proxy
Azure function app that fetches a URL and returns its contents. Used when CloudAPK is blocked from fetching a web resource. For example, see the following bugs in which the Android CloudAPK project was blocked from fetching resources:

- https://github.com/pwa-builder/PWABuilder/issues/1338
- https://github.com/pwa-builder/PWABuilder/issues/1313
- https://github.com/pwa-builder/PWABuilder/issues/1312
- https://github.com/pwa-builder/PWABuilder/issues/1299
- https://github.com/pwa-builder/PWABuilder/issues/1266
